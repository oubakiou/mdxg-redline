# Accessibility（アクセシビリティ）

このセクションは、キーボードからの操作可能性を規定する。

## 13. Keyboard Navigation（キーボードナビゲーション）

[MUST] 準拠ツールは、すべてのコアナビゲーション機能についてキーボードベースの操作をサポートする。

### 13.1 要件

- [MUST] ページナビゲーションのコントロール（例: ページリスト）は矢印キーでキーボードナビゲート可能である
- [MUST] キーボードでページを選択する操作（例: Enter）はそのページへナビゲートする
- [SHOULD] 現在アクティブなページは、ページナビゲーションが発生した時点でフォーカスを受け取る
- [SHOULD] 逐次ナビゲーション（前 / 次）はキーボード経由でアクセス可能である
- [MUST] すべてのインタラクティブ要素（ボタン、リンク、コントロール）は、`aria-label`、可視テキスト、もしくは同等の手段によるアクセシブル名を持つ

### 13.2 実装例

- 矢印キーナビゲーションのための `keydown` ハンドラを持つ、ページリストアイテムへの `tabindex="0"`
- アイコンのみのボタンへの `aria-label` 属性
- ページナビゲーション後にコンテンツ領域へフォーカスを移すフォーカス管理
